Bienvenido al OVI Git y GitHub aquí encontraras la información y el material de apoyo para empezar a trabajar con GitHub la herramienta que usaras para almacenar o alojar tus repositorios del software creado ya sea por medio de git o haciendo uso de un editor de código como los es visual studio code.



Objetivos del sitio:

General: Aclarar e identificar los conceptos y etiquetas básicas que se deben tener en cuenta para la realización de un sitio web.

Objetivo específico 1: Crear un contenido diferente que llame la atención de los estudiantes y que les brinde una información clara y concisa.

Objetivo específico 2:Exponer a través de textos informativos, videos e imágenes lo relacionado con el uso de las herramientas Git y GitHub

Objetivo específico 3:Aclarar las diferentes dudas e inquietudes que presenten estudiantes en su formación a través de diferentes materiales informativos.
