Visual Studio Code

VS Code es un editor de código fuente sofisticado que admite muchas funcionalidades practicas al momento de trabajar con el código.

Caracteristicas: El código combina la interfaz de usuario optimizada de un editor moderno con asistencia y navegación de código enriquecido y una experiencia de depuración integrada, sin la necesidad de un IDE completo.
